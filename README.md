# SHARMA INDUSTRIES - FINANCIAL PERFORMANCE  DASHBOARD
This project presents an interactive Power BI dashboard designed to analyze the financial performance of Sharma Industries for FY 2024–25.
It converts raw transactional data into meaningful insights across revenue, cost, region, and product performance.

**🎯 Problem Statement**
Traditional financial reporting is:
   Time-consuming
   Static (no interactivity)
   Lacks real-time insights
This dashboard solves these issues by enabling dynamic and visual analysis.

**📊 Dashboard Features**
📈 Monthly Revenue Trend
📊 Revenue & Profit by Region
📉 Q3 vs Q4 Cost Analysis (COGS)
📋 Product Profitability Summary
🎛️ Interactive Filters (Year, Region, Quarter)

**📌 Key Insights**
⚠️ Q4 costs are higher than Q3 → Indicates cost pressure
🏆 South Delhi is the top-performing region (~₹91M revenue)
📈 Revenue shows fluctuating monthly trends
📉 Some product categories have low profit margins

**💼 Business Impact**
Reduced manual reporting effort
Improved decision-making speed
Better visibility of financial performance
